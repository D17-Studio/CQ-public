# CQ-public 开发中音游的核心代码

> 本项目是代号 **CQ** 的音游的核心源代码，游戏本身为**非商业项目**，仅供学习、交流与展示。  
> 代码开源旨在分享架构设计与开发经验，欢迎感兴趣的朋友参考、讨论。

---

## ⚠️ 重要说明

- **本仓库仅包含 C# 源代码**，不含 Unity 场景、预制体、美术资源及音频文件，因此**无法直接在 Unity 编辑器中运行**。
- 部分代码依赖 **CRIWARE ADX** 等商业/第三方插件，由于插件本身无法随仓库开源，相关调用在编译时会产生错误。这些代码主要展示音频系统的设计思路，实际运行时需要自行导入对应 SDK。
- 本项目**未使用任何传染性开源协议（如 GPL）**，仅开放源码，不包含完整游戏资源。

---

## 🎯 开源目的

- 展示项目的**代码架构**与**设计模式**
- 分享在音游开发中积累的经验（节奏判定、音频管理、输入处理等）
- 为其他开发者提供参考
